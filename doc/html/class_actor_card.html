Detailed Description

Класс ActorCard - карта-персонаж

Класс ActorCard, преставляющий собой карту-персонаж. Пока значения "здоровья" и "энергии" находятся в пределах допустимых, карта может атаковать другие неограниченное число раз. Может иметь несколько доступных действий, которые можно выбрать.
